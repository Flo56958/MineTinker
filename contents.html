Elevators

 Elevators are a easy way to connect floors in a building.
They require only a Motor and a Block placed on top of the Motor to function.
Since a17 the Elevators are a standalone plugin found here.

Builderswands

Builderswands are a powerful and easy way to place many blocks of the same kind fast. Just have the block you want to place in the inventory and start right-clicking the same block to place one next to it. Builderswands have 5 different tiers and work upto a 5x5-area.

Easy Harvest

Farm crops more easily by right-clicking them with your MineTinker-Hoe.

Automatic replant

Replants crops automatically if the required items are in the inventory.

Commands

  1. mt addexp [amount]: adds XP to the Tool (permission: minetinker.commands.addexp / default: op)
  2. mt help: gives a list of available commands (permission: minetinker.commands.help / default: true)
  3. mt info: gives a short summary about the plugin (permission: minetinker.commands.info / default: true)
  4. mt mods: gives a list of available modifiers (permission: minetinker.commands.modifiers / default: true)
  5. mt name [name]: names the Tool (compatible with color codes! just use '^' instead of 'ยง') (permission: minetinker.commands.name / default: true)
  6. mt addmod [modifier]: adds the modifier to the held tool without using a slot (you need to use the custom names if you have any) (permission: minetinker.commands.addmod / default: op)
  7. mt removemod [modifier]: removes the modifier on the held tool (permission: minetinker.commands.removemod / default: op)
  8. mt setdurability [durability]: sets the durability of the held tool to that specific amount ("full" for full durability) (permission: minetinker.commands.setdurability / default: op)
  9. mt reload: reloads the MineTinker-config (permission: minetinker.commands.reload / default: op)
  10. mt give [ToolType] [level]: gives you a MineTinker-Tool (permissions: minetinker.commands.give / default: op)
  11. mt convert [level]: converts a tool to a MineTinker-Tool (permission: minetinker.commands.convert / default: op)
  12. mt checkupdate: checks if there is a new version on spigotmc.org (permissions: minetinker.commands.checkupdate / default: op)
  13. mt givemodifieritem [Modifier] [Amount] [Player]: gives the player the specified modifier-item (permissions: minetinker.commands.givemodifieritem / default: op)

Permissions

TOOL:

minetinker.tool.create: Allows to create / craft a MineTinker-Tool

minetinker.tool.upgrade: Allows to upgrade a MineTinker-Tool

minetinker.tool.repair: to repair a MineTinker-Tool in the Inventory

MODIFIERS:

minetinker.modifiers.XXX.apply: Allows to apply XXX to a MineTinker-Tool

minetinker.modifiers.XXX.craft: Allows to craft XXX-modifier item (only for crafting through bookshelf)

minetinker.modifiers.XXX.use: Allows to use XXX on a MineTinker-Tool

Example: minetinker.modifiers.autosmelt

ELEVATORS:

minetinker.elevator.use: Allows to use the Elevator of MineTinker

BUILDERSWANDS:

minetinker.builderswands.use: Allows to use the Builderswands of MineTinker

SPAWNER:

minetinker.spawners.mine: Allows to mine a Spawner without breaking it.

minetinker.spawners.place: Allows to place a mined Spawner.

Updater:

minetinker.update.notify: Allows to get notified when MineTinker discovers an update.

FAQ

 Will this plugin be premium when it is finished?

No, it will be free because I don't like to pay for plugins myself. But you can donate via PayPal!

 How can I help with the development?

You can leave a rating on spigotmc.org, or you can report bugs and feature requests in Discord or on Github. If you have experience in coding plugins you can look at the code on Github and help me figure out my mistakes.

 Can you develop a version for 1.XX?

I could, but I will not. I will only develop MineTinker for the most current version of Minecraft (but support from 1.13+). But you can alter the source code for the version you need.
